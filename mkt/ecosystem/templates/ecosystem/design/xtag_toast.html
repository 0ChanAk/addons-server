A toast is a small popup that either provides immediate and non-intrusive feedback on the user's action (email saved), or a notification of an event (new email).

It contains very few text, does not require user interaction, and automatically disappears after a few seconds. It can include text, image, or both.

{% endblock %} {% block use_it %} When the information you want to present is, or corresponds to a set of unique images, i.e. image folders. {% endblock %} {% block tips %}

Do's

  • use very little text

Dont's

  • don't use it to provide information irrelevant to the user's activity at that moment
  • don't use it when action is required. Use an alert or select list instead.
  • if explanation is necessary, use a short text label
