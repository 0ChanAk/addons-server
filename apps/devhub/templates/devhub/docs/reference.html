API & Language References

The official documentation on APIs and languages. What you need, when you need it, to create your masterpiece.

XUL: Defining User Interfaces

  • XUL Overview via Mozilla Developer Center

    XUL (XML User Interface Language) is Mozilla's XML-based language that lets you build feature-rich cross platform applications that can run connected or disconnected from the Internet.

  • XUL Reference via Mozilla Developer Center

    A list of all available XUL elements.

  • XUL Controls List via Mozilla Developer Center

    A description of many available XUL controls, including screenshots.

JavaScript: Specifying Behavior

  • JavaScript Overview via Mozilla Developer Center

    JavaScript is a small, lightweight, object-oriented, cross-platform scripting language.

  • Gecko JavaScript Language Reference via Mozilla Developer Center

    Mozilla Developer Center's JavaScript documentation, including functions, statements, properties, and more.

  • JavaScript Style Guide via Mozilla Developer Center

    Mozilla's JavaScript coding standards.

CSS: Adding Visual Effects

  • CSS Overview via Mozilla Developer Center

    Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML and is used to style a XUL-based application's user interface.

  • Gecko CSS Reference via Mozilla Developer Center

    Mozilla Developer Center's CSS documentation, listing all available properties, values, and selectors.

  • W3C CSS Homepage via World Wide Web Consortium

    Find CSS news, specifications, and tools at the W3C's CSS homepage.

FUEL: Easier Extension Development

  • FUEL API via Mozilla Developer Center

    FUEL is a JavaScript library designed to help developers build extensions using terminology and interfaces that are familiar to them.

  • STEEL API via Mozilla Developer Center

    STEEL is the Scriptable Thunderbird Easy Extension Library. It is a FUEL-like set of interfaces to facilitate Thunderbird extension development.

  • SMILE API via Mozilla Developer Center

    SMILE is a FUEL-like set of interfaces to facilitate SeaMonkey extension development.

XPCOM: Cross-Platform Components

  • XPCOM Overview via Mozilla Developer Center

    XPCOM is a cross-platform component object model with multiple language bindings, letting the XPCOM components be used and implemented in JavaScript, Java, and Python in addition to C++.

  • XPCOM Reference via Mozilla Developer Center

    Mozilla Developer Center's XPCOM API documentation, listing frozen interfaces and functions.

DOM: Interacting with Content

  • DOM Overview via Mozilla Developer Center

    The Document Object Model (DOM) is an API for HTML and XML documents. It provides a structural representation of the document, enabling you to modify its content and visual presentation.

  • Gecko DOM Reference via Mozilla Developer Center

    Mozilla Developer Center's DOM API documentation
